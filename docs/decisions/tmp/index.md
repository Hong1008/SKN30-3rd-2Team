## 후속 블로그 정리 시 사용할 관점

- 각 카드는 **문제 → 선택 → 이유 → 기각/보류 대안 → 결과 상태** 순으로 읽는다.
- 동일 결정을 뒤집거나 확장한 카드도 의도적으로 보존했다. 예: D11→D18, D19→D52, D35→D58, D73/D75→D76.
- 구현 학습은 카드의 영향 영역을 출발점으로 하되, 이 임시 정리에서는 구현의 좋고 나쁨을 평가하지 않는다.
