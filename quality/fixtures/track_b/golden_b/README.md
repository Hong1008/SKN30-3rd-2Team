# Track B 공개 샘플 정책

Track B는 문서 단위 범위 판별과 강건성 점검을 위한 **공개 가능한 샘플**만 사용합니다. 실제 사용자
업로드 문서, 개인 식별 정보가 남은 계약서, 이용·재배포 조건을 확인하지 못한 자료는 fixture로
커밋하지 않습니다.

## 활성 fixture

`scope_manifest.json`에 등록된 원본만 활성 fixture입니다. 각 문서는 공개 표준 양식, 공개 법령
또는 팀이 만든 합성 예제로 한정하며, 용도는 manifest의 `purpose`로 기록합니다. CI와 범위 판별
테스트는 이 목록만 읽습니다.

## 연구 샘플

`scope_manifest.json`에 등록되지 않은 `raw/` 파일은 **연구 샘플**입니다. 자동 평가·CI·배포에
사용하지 않으며, 활성 fixture로 승격하려면 다음을 먼저 기록해야 합니다.

1. 공개 출처 또는 팀 소유·합성 근거
2. 사용 목적
3. 익명화 여부와 개인 식별 정보 검토 결과
4. `scope_manifest.json` 등록과 리뷰

샘플 반입 전에는 파일명과 본문에서 개인 식별 정보를 제거합니다. 사용자 업로드 파일은
`data/99_uploads/`의 일시 처리 대상이며 Git에 커밋하지 않습니다.
