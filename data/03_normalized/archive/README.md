# 표준계약서 구버전 archive

이 폴더는 현재 검토 기준에서는 제외하지만, 개정 이력·기존 골든셋 재현을 위해 Git으로 보관하는
정규화 표준조항과 서브청크입니다.

- `src/pipe/0.migrate.py`는 `03_normalized/` **루트만** 읽으므로 이 폴더의 JSON은 SQLite·Chroma에
  적재되지 않습니다.
- 현재 활성 코퍼스는 계약유형별 최신 표준판 하나만 둡니다. 구버전으로 평가·검토하는 기능은 별도
  사인오프와 명시적 활성화 경로가 필요합니다.
- 새 표준판을 추가할 때는 기존 활성판을 이 폴더로 옮긴 뒤 `just build-db`로 재생성합니다.
